# IoT-Weather-Station

# IoT Μετεωρολογικός Σταθμός

Ένα απλό έργο Internet of Things (IoT) με χρήση του Raspberry Pi Pico W και της MicroPython.

## Περιγραφή

Το έργο μετρά τη θερμοκρασία και την υγρασία του περιβάλλοντος χρησιμοποιώντας τον αισθητήρα **DHT22** και εμφανίζει τις μετρήσεις σε μια οθόνη **LCD 16x2**.

Ένα **RGB LED** αλλάζει χρώμα ανάλογα με τη θερμοκρασία, δίνοντας μια οπτική ένδειξη του επιπέδου άνεσης.

Το έργο δημιουργήθηκε για εκπαιδευτικούς σκοπούς και αποτελεί ένα παράδειγμα χρήσης αισθητήρων και περιφερειακών με τον **Raspberry Pi Pico W**.

## Υλικά

- Raspberry Pi Pico W
- Αισθητήρας θερμοκρασίας και υγρασίας DHT22
- Οθόνη LCD 16x2
- RGB LED
- Breadboard
- Καλώδια σύνδεσης
